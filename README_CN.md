# AI_Capabilities_Evaluation
## 介绍
  AICE 项目 (AI'Capabilities Evaluation Project)是一个有意义的项目，重点评估人工智能各方面的综合能力。

该项目旨在让人们深入了解人工智能在各个领域的能力和未来发展。

科学的方法论是建设这一项目最重要的任务。
## AI 领域 & 能力评估图

### 图像

### 语音
